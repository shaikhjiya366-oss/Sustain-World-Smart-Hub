# 🌍 Sustain World: The Smart Hub

## 📌 Overview
A smart city project integrating sustainability, automation, and user engagement using embedded systems.

## 🚀 Key Features
- ⚡ Footstep Energy Generation using Piezo Sensors
- 🗑 Smart Dustbin with Automatic Opening & Reward System
- 🎮 Interactive Memory Game for Children
- 📢 Smart Advertisement Display System

## 🛠 Technologies Used
- Arduino / ESP32
- Ultrasonic Sensors
- IR Sensors
- Piezoelectric Sensors
- LCD Display
- Embedded C Programming

## ⚙️ System Modules

### 1. Energy Generation
- Converts footsteps into electrical energy
- Stores energy in batteries

### 2. Smart Dustbin
- Detects user using ultrasonic sensor
- Opens automatically
- Dispenses reward (water/cold drink)

### 3. Interactive Game
- Memory-based LED game
- Enhances child learning

### 4. Advertisement System
- Displays ads dynamically
- Can be updated remotely

## 📊 Results
- Generated ~1.5V–3.5V per step
- Fully automated dustbin system
- Working prototype of smart hub
